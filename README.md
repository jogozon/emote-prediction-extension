##General info
This is project is a Chrome Extension that uses NLP to analyze chat messages on Joe Bartolozzi's Twitch stream and suggest an appropriate emote to type in the chat for the user. 

##Technologies
Project is created with:
* Flask
* Java Script
* Python
* Tailwind
* HTML
